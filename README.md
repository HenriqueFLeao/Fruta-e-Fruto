# Fruto e Fruta 


Mais um projeto para o Portfólio, desenvolvido como parte de um curso da plataforma Alura, o objetivo de exibir o meu aprendizado permitindo que outras pessoas acompanhem minha evolução nesse mundo de desenvolvedor Web. Aqui está uma visão geral dos recursos e tecnologias utilizadas neste projeto.

## Proposta

A proposta do curso dessa vez foi aprender boas normas de nomeação de arquivos Style e nomenclaturas de classes etc...,para aprender isso foi proposto fazer um site de receitas simples com base de HTML e CSS apenas com o front-end funcionando separando cada classe em um arquivo .css diferente, colocando em pastas organizadas 

## Recursos Principais

- *Site*: Site simples feito somente em HTML e CSS com header interativo ao passar o cursor do mouse por cima das opções
  
- *Responsividade*: A responsividade não era a prioridade desse projeto porém ele tem sim responsividade para mobile e Desktop
